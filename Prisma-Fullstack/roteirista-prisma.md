# Roteirista Cinematográfica — 8 Reels para Redes Sociais

Cada Reel é uma peça cinematográfica curta.
Não é vídeo gravado no celular sem pensar — é direção com intenção.
O seguidor não percebe a técnica. Só sente que não consegue parar de assistir.

---

## Princípios cinematográficos para redes sociais

### O Reel não é uma palestra gravada
É uma experiência visual + verbal simultânea.
A câmera conta tanto quanto o que é dito.

### Regra do corte emocional
Corta quando a emoção está no pico — não depois.
O corte no momento certo é o que mantém o ritmo.

### Regra da tela cheia
Todo Reel é assistido em formato vertical, tela cheia.
O rosto e os olhos ocupam o centro — a expressão é parte do roteiro.

### Regra do silêncio
Uma pausa de 0,5 segundo antes de uma frase importante
faz essa frase soar três vezes mais forte.

---

## Formatos e durações

```
15 segundos — impacto puro
→ Gancho + 1 ponto central + CTA
→ Zero desperdício de segundo
→ Ideal para Reels de dor e contraste

30 segundos — equilíbrio
→ Gancho + 2-3 pontos + CTA
→ O formato que mais performa em alcance
→ Ideal para ensinamento e autoridade

60 segundos — profundidade
→ Gancho + desenvolvimento completo + CTA
→ Exige ritmo acelerado de cortes
→ Ideal para prova social e propósito
```

## Estrutura cinematográfica por Reel

### Reel 1 — Dor Nomeada (30s recomendado)
```
CENA 1 (0–3s): Rosto próximo, olhar direto para câmera
FALA: [gancho de identificação imediata]
CORTE

CENA 2 (3–12s): Mesmo enquadramento ou ligeiro afastamento
FALA: [desenvolvimento da dor — sem solução]
CORTE RÍTMICO a cada 2-3 segundos

CENA 3 (12–25s): Reaproximação — intimidade
FALA: [nomeação precisa do que o seguidor sente]
PAUSA intencional antes da última frase

CENA 4 (25–30s): Olhar direto, tom de convite
FALA: [CTA natural — sem forçar]
```

### Reel 2 — Contraste de Mercado (30s recomendado)
```
CENA 1 (0–3s): Câmera levemente afastada — presença de palco
FALA: [dado ou observação que provoca]
CORTE

CENA 2 (3–20s): Sequência de cortes rápidos (a cada 2s)
FALA: [argumentação em blocos curtos]
Cada corte reforça o ponto anterior

CENA 3 (20–30s): Câmera próxima — intimidade de conclusão
FALA: [virada + CTA]
```

### Reel 3 — Apresentação da Solução (30-60s)
```
CENA 1 (0–3s): Resultado primeiro — não apresentação
FALA: [o que é possível — antes de "quem sou eu"]
CORTE

CENA 2 (3–30s): Apresentação por evidência, não por currículo
FALA: [o que faço, para quem, com qual resultado]
CORTES a cada ponto

CENA 3 (30–60s): Convite
FALA: [CTA — próximo passo claro]
```

### Reel 4 — Bastidor Real (30-60s)
```
CENA 1 (0–3s): Ambiente real — não estúdio
FALA: [contextualização do momento]
CORTE

CENA 2 (3–45s): Processo em tempo real ou narrado
FALA: [o que está acontecendo + por quê importa]
CORTES naturais — não mecânicos

CENA 3 (45–60s): Reflexão + convite
FALA: [conexão com o seguidor + CTA]
```

### Reel 5 — Ensinamento Prático (30-60s)
```
CENA 1 (0–3s): Promessa de valor imediato
FALA: "[1 coisa que você pode fazer hoje para...]"
CORTE IMEDIATO

CENA 2 (3–50s): Passo a passo em blocos
FALA: Cada bloco = 1 instrução clara
CORTE entre cada instrução
B-roll ou texto em tela para reforçar

CENA 3 (50–60s): Síntese + CTA
FALA: [resumo em 1 frase + próximo passo]
```

### Reel 6 — Autoridade com Opinião (30s recomendado)
```
CENA 1 (0–3s): Opinião forte — câmera próxima
FALA: "[afirmação contraintuitiva ou polêmica produtiva]"
SEM hesitação — tom seguro

CENA 2 (3–25s): Argumento — ritmo acelerado
FALA: [evidências em blocos de 2-3 segundos]
CORTES no pico de cada argumento

CENA 3 (25–30s): Conclusão e convite
FALA: [reafirmação + CTA]
```

### Reel 7 — Resultado de Cliente (60s recomendado)
```
CENA 1 (0–3s): Resultado em destaque — número ou transformação
FALA: "[resultado específico e mensurável]"
CORTE

CENA 2 (3–40s): Jornada — de onde veio até onde chegou
FALA: [narrativa da transformação — cronológica]
CORTES a cada capítulo da história

CENA 3 (40–55s): Depoimento real (se disponível) ou narração
FALA: [voz do cliente ou narração do profissional]

CENA 4 (55–60s): Convite para quem quer o mesmo
FALA: [CTA — identificação + próximo passo]
```

### Reel 8 — Propósito e Manifesto (60s recomendado)
```
CENA 1 (0–5s): Ambiente significativo — não neutro
FALA: [momento de origem — o que mudou tudo]
RITMO lento — pausa permitida

CENA 2 (5–45s): Propósito em camadas
FALA: [por que faço o que faço — sem roteiro de palco]
CORTES espaçados — respira mais que os outros Reels

CENA 3 (45–60s): Manifesto + convite
FALA: [se você acredita nisso, você está no lugar certo]
TOM: íntimo, não grandioso
```

## Direção técnica

### Enquadramentos recomendados
```
Close (rosto + ombros) → emoção, intimidade, opinião
Médio (busto) → ensinamento, apresentação
Aberto (corpo inteiro) → contexto, bastidor, autoridade de palco
```

### Iluminação
```
Luz natural de janela lateral → bastidor, propósito
Ring light frontal → ensinamento, autoridade
Luz ambiente quente → origem, manifesto
```

### Trilha
```
Sem trilha → depoimento, propósito, manifesto (voz tem que ser ouvida)
Trilha baixa → ensinamento, apresentação
Trilha em destaque → abertura de impacto (primeiros 3s)
```

### Legenda
```
Sempre ativa — 85% dos Reels são assistidos sem som
Fonte grande, contraste alto, posição central-inferior
Sincronizada com o ritmo de fala — não com o texto completo
```
