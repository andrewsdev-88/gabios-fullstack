# Orquestradora Etapa 05 — Relatório Final + Ciclo de Melhoria (Fase 04)

## Propósito

Encerrar o ciclo PRISMA com rigor. Métricas reais, ROI documentado,
espelho do sistema e abertura limpa para o próximo ciclo.

```
Etapa 04 → [ ETAPA 05 — RELATÓRIO FINAL ] → próximo ciclo (orq-etapa-00)
```

Obrigatória em todo ciclo, independente do plano contratado.

---

## Input obrigatório

| Input | Origem |
|-------|--------|
| Métricas e dados de performance do cliente | Glenda coleta e traz |
| Todos os outputs das etapas anteriores | orq-etapa-01 a 04 |
| BDP aprovado | orq-etapa-02 |

---

## O que esta etapa entrega

- Relatório Final com métricas, ROI e análise de crescimento
- Espelho do sistema (PDCA do PRISMA aplicado a este ciclo)
- Direção para o próximo ciclo
- Relatório 04 com abertura de loop para continuidade (sem oferta direta)

---

## Pipeline de Execução

### PASSO 1 — COLETA DE MÉTRICAS

Identificar com a Glenda:
- Crescimento de seguidores no período
- Taxa de engajamento (antes × depois)
- Consultas ou leads gerados via digital
- Conversões atribuíveis ao conteúdo
- ROI calculado sobre o investimento no PRISMA
- O que funcionou, o que não funcionou

### PASSO 2 — ESPELHO DO SISTEMA

Acionar **espelho-prisma**:
- Input: métricas coletadas + todos os outputs do ciclo
- PDCA aplicado ao sistema:
  - PLAN: o que foi projetado (Vértice + Revelação)
  - DO: o que foi entregue (Studios + conteúdo)
  - CHECK: gap entre projetado e entregue
  - ACT: ajustes no sistema para o próximo ciclo
- Output: diagnóstico do ciclo + recomendações de ajuste

### PASSO 3 — RELATÓRIO FINAL (RELATÓRIO 04)

Acionar **apresenta-prisma**:
```
[MODO_DE_EXECUCAO]: SYSTEM_DESIGN
```
- Inclui: métricas, ROI, antes × depois, direção para o próximo ciclo
- Seção final: abertura de loop para continuidade sem oferta direta
- Design System: BDP do cliente
- G Gabi Produções aparece apenas como assinatura no rodapé

### PASSO 4 — ABERTURA DO PRÓXIMO CICLO

Ao encerrar:
- Registrar aprendizados do ciclo
- Atualizar contexto do cliente para o próximo ciclo
- Acionar **orq-etapa-00** com contexto renovado

---

## Mentes desta etapa

| Mente | Função |
|-------|--------|
| espelho-prisma | PDCA do sistema. Diagnóstico do ciclo e recomendações. |
| apresenta-prisma | Relatório 04 com métricas, ROI e abertura de loop |

---

## Regras

- Obrigatória em todo ciclo — independente do plano.
- Nunca encerrar sem o espelho-prisma rodado.
- Métricas são reais — nunca projetadas ou estimadas sem base.
- O Relatório 04 não faz oferta direta — abre loop para a Glenda conduzir.
- O próximo ciclo começa com contexto enriquecido do cliente.
